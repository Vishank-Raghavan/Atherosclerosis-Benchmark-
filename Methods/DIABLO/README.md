# DIABLO Analysis

Multi-omics integration using DIABLO method from mixOmics package.

## Run

```bash
Rscript DIABLO_Analysis.R
```

## Outputs

- `diablo_plots.pdf` - Visualizations
- `diablo_selected_proteins.csv` - Selected proteins

## Method

DIABLO (Data Integration Analysis for Biomarker discovery using Latent cOmponents) integrates multiple omics datasets while identifying discriminative features.

Reference: Singh et al. (2019) Bioinformatics 35(17):3055-3062
